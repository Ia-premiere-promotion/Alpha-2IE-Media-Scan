# Logos des Partenaires

Placez les logos des partenaires dans ce dossier avec les noms suivants :

- `csc-logo.png` - Conseil Supérieur de la Communication
- `mtdpce-logo.png` - Ministère de la Transition Digitale
- `citadel-logo.png` - CITADEL
- `lefaso-logo.png` - Lefaso.net
- `fasopresse-logo.png` - FasoPresse
- `sidwaya-logo.png` - Sidwaya
- `observateur-logo.png` - L'Observateur Paalga
- `aib-logo.png` - AIB (Agence d'Information du Burkina)

## Recommandations
- Format : PNG avec fond transparent de préférence
- Dimensions recommandées : 300x200px maximum
- Poids : < 100KB par image
